但是在这个本垒打的棒球当中寄宿了父亲的魂，由此主人公与其“棒球父亲”的故事开始了。<br>
主人公的目标是获得全国大赛的优胜，而在游戏中的棒球部分和系列以往稍微有些不同，采用了“卡片棒球”系统，这让不擅长动作游戏的人也能享受到该游戏的乐趣。</p>
